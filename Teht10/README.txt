Github API ASP.NETissä - Octokit

1.Octokit asennus
	- Nuget package managerilla

2.Tietoa järjestelmästä
	- Octokitillä on helppo hakea tavaraa Githubin API:sta

3.Järjestelmävaatimuksena
	- Visual Studio

4.Ohjelmiston mukana tulevat tiedostot
	- Octokit.dll

5.Oletushakemistojen asettaminen
	- Ei ole

6.Huomioitavaa
	- OAuthin tai basic authin avulla pääsisi myös tekemään muokkaus/lisäys/poisto toimintoja
	- Octokit on Githubin itsenä ylläpitämä, löytyy .NETin lisäksi Rubylle ja Objective-C:lle

7.Tiedossa olevat ongelmat
	- Githubin API:in voi tehdä vain 60 pyyntöä tunnissa samasta IP:stä

8.Mitä opittu?
	- Octokitin käyttö, asynkronisuus

9.Tekijä
	- Wynn Netherland, Adam Stacoviak, Erik Michaels-Ober
